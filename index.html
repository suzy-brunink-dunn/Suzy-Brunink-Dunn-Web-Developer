Portfolio

Recipe Cards

These fun recipe cards represent an early static website project I completed through my online courses with Skillcrush. Here I practiced my basic CSS layout skills using and clearing floats, applying the overflow property, and creating images with a static height and width. Since CSS Grid and Flexbox, this is now considered a legacy layout method but one still worth understanding as an important building block.

Rogue Pickings

This single page site for a fictional food delivery service was created by pulling all design elements from a PSD file. It then pushed my HTML5 and CSS3 skills up a notch by applying flexbox properies and media queries to make it fully responsive across mobile, tablet and desktop devices.

Cuppa Joe

I found the bones of this project through a course on building responsive websites and turned the original project into the fictional coffee shop of my dreams. Here I worked on using responsive CSS and media queries while also managing to give the site a little personality.

Unplugged

Unplugged, a fictional tech industry nature retreat, was converted from PSD files to create a mobile first responsive website. This site utilizes an HTML5 Boilerplate shell and features multiple pages, a form for user input, embedded iframe, and responsive typography.

About

Let me introduce myself...

My name is Suzy and I'm a self-taught responsive designer & developer who thrives on visual storytelling and the best user experience. After earning an advanced degree in Film Studies from New York University, my professional roles have been mainly focused in the content and media management worlds.

However, I have always held a strong passion for technology and my love of coding and user design has proven to be the ideal partner to my creative background. When I'm not obsessing over fonts or working, I can usually be found walking my precocious senior terrier, watching old movies, hikingbikingcanoeing, or sipping a fine old-timey cocktail on the rocks.
